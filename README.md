# MR Hands-on

Marker-based augmented reality that runs in a phone browser. No app, no headset.

Everything you need — setup, the stages, and the exercises — is on the hub page.
Fork this repository, turn on GitHub Pages (Settings → Pages → Deploy from a
branch → `main` → `/ (root)`), and open your copy:

```
https://<your-github-username>.github.io/<repository-name>/
```

Built with [A-Frame](https://aframe.io) and [MindAR](https://hiukim.github.io/mind-ar-js-doc/).
